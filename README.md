# Visual Listening

## Description
Computer vision project that assumes datapoints of demographics used by marketing companies. 

## Concept
This project explores the idea of visual listening through online imagery to generate data on users based on demographics, to help brands to target to them.

## Next steps
Firstly, i will use a Python API called DeepFace which generates demographics like age, gender, and ethnicity, with the possibility to train other categories. This will give users a shocking representation of how the internet and marketers see them. 
Secondly, the hitpoints will be improves to 'targets' used in flocking autonomous agents, so when the user moves around, the data is less attached to them, but unavoidably following them. 

## Instructions (operation manual)
This static website uses ml5. A functioning internet connection is required. To move the datapoints, stand back and move around - you are the controller.
